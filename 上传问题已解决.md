## 注意事项

### 添加新文章时：
确保包含以下字段：
```javascript
{
  title: "文章标题",
  tag: "紧急预警",           // 标签文本
  tagType: "danger",         // danger/warning/info
  category: "刷单返利",      // 分类（必须与前端分类名称一致）
  content: "文章内容",
  timestamp: Date.now(),     // 时间戳
  views: 0,
  author: "反诈小助手"
}
```

### 如果新文章缺少字段：
运行云函数自动添加：
- 缺少 `tag` 和 `tagType`：运行 `updateArticleTags`
- 缺少 `category`：运行 `updateArticleCategories`
