## 使用指南

### 查看用户手册

1. 进入 `user-manual/` 目录
2. 选择对应模块的用户手册文件
3. 按章节阅读操作指南

### 执行测试案例

1. 阅读 `test-cases/TEST_PLAN.md` 了解测试计划
2. 选择对应模块的功能测试案例文件
3. 按照测试案例逐条执行
4. 记录测试结果

### 测试案例命名规范

- `TC-[模块编号]-[序号]` - 功能测试案例
- `IT-[序号]` - 集成测试案例
