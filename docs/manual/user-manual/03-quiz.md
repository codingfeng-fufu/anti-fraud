# 趣味答题 - 用户手册

## 功能概述

趣味答题是反诈卫士的核心学习功能，通过随机抽题、即时判分、排行榜竞技的方式，让用户在答题过程中学习反诈知识。

## 入口路径

- 点击底部导航栏"答题"图标
- 从其他页面跳转进入

## 界面说明

### 答题主页元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 开始答题按钮 | 页面中央 | 开始新一轮答题 |
| 排行榜入口 | 页面下方 | 查看答题排行榜 |
| 历史成绩 | 页面顶部 | 显示答题统计信息 |

| 元素 | 说明 |

| 上一题按钮 | 返回上一题（可修改答案） |

## 操作流程

### 开始答题

1. 进入答题页面
2. 点击"开始答题"按钮
3. 系统随机抽取10道题目
4. 进入答题界面

### 答题过程

1. 阅读题目内容
2. 点击选择你认为正确的答案
3. 选中的答案高亮显示
4. 点击"下一题"继续
5. 如需修改，点击"上一题"返回

### 提交答卷

1. 完成所有10道题目
2. 最后一题显示"提交"按钮
3. 点击"提交"
4. 系统自动判分

### 查看结果

1. 提交后显示结果页面
2. 显示得分和正确题数
3. 显示获得的积分：
   - 排行榜积分：每答对1题+1分（累计）
   - 商城积分：每日上限30分（北京时间UTC+8重置）
4. 可查看每道题的正确答案和解析

### 查看排行榜

1. 点击"排行榜"按钮
2. 进入排行榜页面
3. 查看积分排名前100的用户
4. 显示自己的排名位置

## 积分规则

### 排行榜积分

- 每答对1题获得1分
- 累计计算，不清零
- 用于排行榜排名

### 商城积分

- 每答对1题获得1分商城积分
- **每日上限30分**（北京时间UTC+8）
- 超过上限后答对不再获得商城积分
- 次日0点重置上限

### 积分示例

| 场景 | 排行榜积分 | 商城积分 |
|-----|----------|---------|
| 今日首次答题对5题 | +5 | +5 |
| 今日第二次答题对8题 | +8 | +8 |
| 今日第三次答题对10题 | +10 | +17（已达30分上限）|
| 明日答题对10题 | +10 | +10 |

## 注意事项

- 每轮答题共10道题
- 答题过程中不能退出，退出视为放弃本轮
- 答案提交后不可修改
- 题目随机抽取，可能重复
- 网络不稳定时可能加载失败，请重试
