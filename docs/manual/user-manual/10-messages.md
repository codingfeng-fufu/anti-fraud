# 站内消息 - 用户手册

## 功能概述

站内消息功能用于接收系统通知，包括关注用户的新帖通知、评论回复通知、系统公告等。

## 入口路径

- 个人中心 → 站内消息
- 或点击个人中心的消息红点图标

## 界面说明

### 页面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 消息列表 | 页面主体 | 显示所有消息 |
| 未读标识 | 消息条目左侧 | 蓝点表示未读 |

### 消息卡片元素

| 元素 | 说明 |
|-----|------|
| 消息类型图标 | 消息类型标识 |
| 消息标题 | 消息摘要标题 |
| 消息预览 | 消息内容预览 |
| 时间 | 消息发送时间 |
| 已读/未读状态 | 未读消息有蓝点标记 |

## 消息类型

| 类型 | 图标 | 说明 |
|-----|------|-----|
| 新帖通知 | 📝 | 关注用户发布了新帖子 |
| 回复通知 | 💬 | 有人回复了你的帖子或评论 |
| 系统公告 | 🔔 | 系统重要通知 |

## 操作流程

### 查看消息列表

1. 进入站内消息页面
2. 自动加载消息列表
3. 向上滑动加载更多消息

### 查看消息详情

1. 点击消息条目
2. 自动标记为已读
3. 跳转到相关页面：
   - 新帖通知 → 帖子详情页
   - 回复通知 → 被回复的内容页

### 刷新消息

1. 下拉页面
2. 等待刷新完成
3. 显示最新消息

## 注意事项

- 消息保留30天，超期自动清理
- 最多显示100条消息
- 点击消息自动标记已读
- 未读消息数量显示在个人中心图标上
