# 积分商城 - 用户手册

## 功能概述

积分商城是用户使用积分兑换商品的功能模块，支持兑换虚拟商品、道具卡和实体商品。

## 入口路径

- 个人中心 → 积分商城
- 或直接点击积分显示区域

## 界面说明

### 页面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 积分余额 | 页面顶部 | 显示当前可用积分 |
| 商品分类标签 | 商品列表上方 | 筛选商品类型 |
| 商品列表 | 页面主体 | 展示可兑换商品 |
| 兑换记录入口 | 页面右上角 | 查看历史兑换 |

### 商品卡片元素

| 元素 | 说明 |
|-----|------|
| 商品图片 | 商品展示图 |
| 商品名称 | 商品名称 |
| 所需积分 | 兑换需要的积分 |
| 库存状态 | 显示剩余库存 |
| 兑换按钮 | 点击进行兑换 |

### 商品分类

| 分类 | 说明 |
|-----|------|
| 全部 | 所有商品 |
| 虚拟商品 | 话费充值、流量包等 |
| 道具卡 | 积分加倍卡等道具 |
| 实体商品 | 实物奖品 |

## 操作流程

### 浏览商品

1. 进入积分商城
2. 默认显示全部商品
3. 点击分类标签筛选商品类型

### 兑换商品

1. 点击商品卡片上的"兑换"按钮
2. 弹出确认框，显示商品信息和所需积分
3. 确认兑换

### 兑换流程分支

#### 虚拟商品/道具卡（无需填写信息）

1. 确认兑换
2. 系统扣除积分
3. 商品自动发放到背包
4. 显示"兑换成功"

#### 需要填写兑换信息的商品

1. 确认兑换
2. 跳转到兑换信息填写页
3. 填写手机号等必要信息
4. 提交兑换申请
5. 等待人工发放

### 查看兑换记录

1. 点击页面右上角"记录"按钮
2. 进入兑换历史页面
3. 查看所有兑换记录和状态

## 兑换规则

### 积分检查

- 兑换前检查积分是否足够
- 积分不足时提示当前积分和所需积分

### 库存检查

- 兑换前检查商品库存
- 库存不足时提示"库存不足"

### 兑换限制

| 商品类型 | 限制 |
|---------|------|
| 虚拟商品 | 自动发放到背包 |
| 道具卡 | 自动发放，可立即使用 |
| 实体商品 | 需填写收货信息，人工处理 |

## 注意事项

- 兑换成功后积分立即扣除，不可撤销
- 实体商品需等待人工处理发货
- 如遇问题请联系客服
- 积分不足可通过签到、答题、阅读等方式获取

## 获取更多积分

点击"获取积分"按钮可查看积分获取方式：
- 完成每日签到
- 阅读反诈文章
- 参与趣味答题
- 与AI助手对话
- 解锁成就奖励
