# AI助手 - 用户手册

## 功能概述

AI助手基于通义千问大模型，提供智能反诈问答服务。支持文字对话和图片识别，能够解答反诈问题、识别可疑信息、提供防骗建议。

## 入口路径

- 点击底部导航栏"AI助手"图标

## 界面说明

### 界面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 对话区域 | 页面上方 | 显示对话消息历史 |
| 欢迎消息 | 对话区顶部 | AI助手功能介绍 |
| 输入框 | 页面底部 | 输入问题内容 |
| 发送按钮 | 输入框右侧 | 发送消息 |
| 图片按钮 | 输入框左侧 | 上传图片进行识别 |
| 清除按钮 | 页面右上角 | 清除对话历史 |

### 消息类型

| 类型 | 样式 | 说明 |
|-----|------|-----|
| 用户消息 | 右侧蓝色气泡 | 用户发送的文字或图片 |
| AI回复 | 左侧白色气泡 | AI助手的回复内容 |
| 图片消息 | 右侧带图片 | 用户上传的图片缩略图 |

## 操作流程

### 文字对话

1. 点击输入框，键盘弹出
2. 输入你的问题
3. 点击发送按钮
4. 等待AI回复
5. 查看AI回复内容

### 图片识别

1. 点击输入框左侧的图片按钮
2. 选择图片来源（相册/拍照）
3. 选择要识别的图片
4. 系统自动分析图片
5. 显示AI分析结果

### 查看历史记录

1. 进入AI助手页面
2. 自动加载本地保存的历史记录
3. 向上滑动查看更早的对话

### 清除历史记录

1. 点击页面右上角菜单按钮
2. 选择"清除历史记录"
3. 确认清空操作
4. 对话记录被清除

## 对话能力

### 支持的功能

| 功能 | 示例问题 |
|-----|---------|
| 反诈知识解答 | "什么是杀猪盘？" |
| 可疑信息识别 | "这条短信是诈骗吗？" |
| 防骗建议 | "如何保护个人信息？" |
| 图片分析 | 上传截图分析诈骗风险 |
| 追问对话 | "那怎么办？"、"还有呢？" |

### 上下文记忆

- AI会记住最近**10轮对话**
- 可以追问相关问题
- AI会基于之前的内容回答

## 注意事项

### 隐私保护

- 对话记录**仅保存在你的手机本地**
- 不会上传到服务器存储
- 清除后无法恢复

### 使用限制

- 每次对话字数上限：2000字
- 图片大小限制：5MB以内
- 支持的图片格式：JPG、PNG
- 对话次数影响积分获取

### 网络要求

- 需要网络连接才能使用
- 网络不稳定时可能回复较慢
- 失败时会显示错误提示

## 紧急情况

如遇紧急诈骗情况：
- **拨打反诈专线：96110**
- AI回复中也会提供紧急联系方式
