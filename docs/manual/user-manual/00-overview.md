# 产品概述 - 用户手册

## 1. 产品简介

**反诈卫士**是一款面向大学生群体的微信小程序，旨在通过趣味化的方式帮助大学生学习反诈知识、提高防骗意识。小程序融合了AI智能助手、趣味答题、社区互动等多种功能，让反诈学习变得生动有趣。

## 2. 产品特色

- **AI智能问答**：基于通义千问大模型，支持文字和图片识别，提供专业的反诈咨询
- **趣味答题系统**：随机抽题、即时判分、排行榜竞技，让学习更有动力
- **社区互动平台**：分享防骗经验，帮助更多人远离诈骗
- **积分激励体系**：签到、答题、学习均可获得积分，可兑换丰富奖品
- **成就与称号系统**：解锁成就，装备个性称号，展示反诈达人身份

## 4. 用户角色

| 角色 | 说明 | 权限 |
|-----|------|-----|
| 游客 | 未登录用户 | 可浏览文章、社区帖子 |
| 普通用户 | 已登录用户 | 全部功能 |
| 已认证用户 | 完成学生认证 | 额外积分奖励、专属功能 |

## 5. 积分规则

### 5.1 积分获取方式

| 行为 | 积分 | 说明 |
|-----|------|-----|
| 每日签到 | 5-50 | 连续签到奖励递增 |
| 阅读文章 | 2/篇 | 每日上限10篇 |
| AI对话 | 2/次 | 每日上限20次 |
| 答题正确 | 1/题 | 排行榜积分，商城积分日上限30 |
| 首次学生认证 | 50 | 一次性奖励 |
| 成就解锁 | 各成就 | 额外积分奖励 |

### 5.2 积分消耗方式

| 行为 | 说明 |
|-----|------|
| 兑换商品 | 在积分商城兑换虚拟/实体商品 |
| 兑换称号 | 兑换特殊称号 |

## 6. 快速上手

### 6.1 新用户引导

1. 打开微信，搜索"反诈卫士"小程序
2. 首次进入自动登录，获取微信头像昵称
3. 建议完成学生认证，获得额外积分奖励
4. 完成首次签到，开启积分之旅

### 6.2 日常使用建议

- **每日签到**：进入"我的"页面点击签到，或直接进入签到日历页
- **学习文章**：在首页浏览反诈资讯，学习防骗知识
- **趣味答题**：每天答一组题，巩固反诈知识
- **AI咨询**：遇到可疑信息，随时咨询AI助手

## 7. 帮助与反馈

- **在线反馈**：设置 → 意见反馈
- **诈骗举报**：我的 → 诈骗反馈
- **紧急求助**：拨打反诈专线 **96110**

## 8. 隐私保护

- 对话记录仅保存在用户手机本地，不上传服务器
- 学生信息加密存储，仅用于身份验证
- 用户可随时清除本地数据
