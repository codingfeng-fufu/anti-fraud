# 学生认证 - 用户手册

## 功能概述

学生认证功能允许用户绑定学生身份，完成认证后可获得额外积分奖励，并解锁专属功能。

## 入口路径

- 个人中心 → 学生认证

## 界面说明

### 表单元素

| 元素 | 类型 | 说明 |
|-----|------|-----|
| 学校名称 | 显示 | 固定显示"中央财经大学" |
| 院系 | 选择器 | 选择所属院系 |
| 年级 | 选择器 | 选择年级 |
| 学号 | 文本输入 | 输入10位学号 |
| 姓名 | 文本输入 | 输入真实姓名 |
| 手机号 | 文本输入 | 输入手机号（可选） |

## 操作流程

### 完成认证

1. 进入学生认证页面
2. 选择所属院系
3. 选择年级
4. 输入学号（10位数字）
5. 输入真实姓名
6. 输入手机号（可选）
7. 点击"提交"按钮
8. 等待验证

### 认证成功

1. 显示"绑定成功"弹窗
2. 首次认证获得50积分奖励
3. 自动返回个人中心

### 解除认证

1. 进入设置 → 账户管理
2. 点击"解除学生认证"
3. 确认解除

## 输入验证

| 字段 | 验证规则 |
|-----|---------|
| 学号 | 必填，10位数字 |
| 姓名 | 必填，2-20个字符 |
| 手机号 | 可选，11位手机号格式 |

## 注意事项

- 学号格式必须为10位数字
- 每个学号只能绑定一个账号
- 首次认证获得50积分奖励
- 解除认证后可重新认证
- 学生信息仅用于身份验证，严格保密

## 隐私保护

- 学生信息加密存储
- 仅用于身份验证
- 不会向第三方泄露
- 用户可随时解除认证
