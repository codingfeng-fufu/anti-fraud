# 学习记录 - 用户手册

## 功能概述

学习记录页面展示用户在反诈卫士中的学习统计数据，包括阅读历史、学习时长、对话记录等。

## 入口路径

- 个人中心 → 学习记录

## 界面说明

### 页面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 学习统计卡片 | 页面顶部 | 显示学习数据概览 |
| Tab切换 | 统计卡片下方 | 切换文章/对话记录 |
| 记录列表 | 页面主体 | 显示历史记录 |

### 统计数据项

| 数据项 | 说明 |
|-------|------|
| 阅读文章数 | 累计阅读文章数量 |
| 学习时长 | 累计学习时间（分钟） |
| AI对话次数 | 累计对话次数 |
| 连续学习天数 | 连续签到天数 |

### Tab页面

| Tab | 内容 |
|-----|------|
| 文章 | 阅读过的文章列表 |
| 对话 | AI对话统计（对话次数） |

## 操作流程

### 查看阅读历史

1. 进入学习记录页面
2. 默认显示"文章"Tab
3. 查看阅读过的文章列表
4. 每条记录显示：文章标题、阅读时间

### 查看对话统计

1. 点击"对话"Tab
2. 查看AI对话次数统计
3. 显示累计对话次数

### 重新阅读文章

1. 在阅读历史列表中
2. 点击文章条目
3. 跳转到文章详情页

### 清空历史记录

1. 点击"清空记录"按钮
2. 确认清空操作
3. 历史记录被清空
4. 统计数据归零

## 数据来源

| 数据 | 来源 |
|-----|------|
| 阅读历史 | 本地存储 |
| 学习时长 | 阅读时长累计 |
| 对话次数 | 云端统计 + 本地缓存 |
| 签到天数 | 云端统计 |

## 注意事项

- 阅读历史保存在本地，清除小程序数据后会丢失
- 最多显示50条阅读记录
- 清空后不可恢复
- 对话次数从云端同步，换设备可恢复
