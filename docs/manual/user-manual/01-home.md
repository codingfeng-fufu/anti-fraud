# 首页功能 - 用户手册

## 功能概述

首页是反诈卫士的资讯中心，提供反诈文章浏览、分类筛选、搜索等功能。用户可以在这里获取最新的反诈资讯、学习防骗知识。

## 入口路径

- 小程序启动后默认进入首页
- 点击底部导航栏"首页"图标

## 界面说明

### 界面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 搜索框 | 顶部 | 输入关键词搜索文章和用户 |
| 轮播图 | 搜索框下方 | 展示重要活动或文章推荐 |
| 分类标签 | 轮播图下方 | 筛选不同类型的诈骗文章 |
| 文章列表 | 页面主体 | 展示反诈文章卡片 |
| 下拉刷新 | 全页 | 更新文章列表 |

### 文章卡片元素

| 元素 | 说明 |
|-----|------|
| 标签 | 文章类型标签（紧急预警/案例分析/防骗知识等） |
| 标题 | 文章标题 |
| 发布时间 | 相对时间显示（刚刚/几分钟前/几小时前/日期） |
| 浏览量 | 文章阅读次数，超过1000显示为1k，超过10000显示为1w |

## 操作流程

### 浏览文章

1. 打开小程序，自动加载文章列表
2. 向下滑动浏览更多文章
3. 点击文章卡片进入文章详情页

### 分类筛选

1. 在分类标签区域点击目标分类
2. 文章列表自动筛选该分类下的文章
3. 当前选中的分类高亮显示

**可用分类：**
- 全部
- 刷单返利
- 虚假投资理财
- 虚假购物服务
- 冒充电商客服
- 虚假贷款
- 虚假征信
- 冒充领导熟人
- 冒充公检法
- 网络婚恋交友
- 网游虚假交易

### 搜索文章

1. 点击顶部搜索框
2. 输入搜索关键词
3. 点击搜索或回车
4. 跳转到全局搜索结果页

### 刷新内容

1. 在首页向下拉动页面
2. 释放后触发刷新
3. 等待加载完成
4. 显示"刷新成功"提示

### 查看轮播图

1. 左右滑动浏览轮播图
2. 点击轮播图可跳转到对应活动页面

## 注意事项

- 文章列表最多显示50篇文章
- 首次加载从云数据库获取，离线时使用本地缓存
- 浏览量会实时更新并同步到云端
- 若分类下无文章，会提示"暂无相关内容"
