# 个人中心 - 用户手册

## 功能概述

个人中心是用户管理个人信息、查看数据统计、访问各项功能的入口页面。

## 入口路径

- 点击底部导航栏"我的"图标

## 界面说明

### 页面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 用户头像 | 页面顶部左侧 | 显示/修改头像 |
| 用户昵称 | 头像右侧 | 显示/修改昵称 |
| 签到按钮 | 顶部醒目位置 | 每日签到 |
| 连续签到天数 | 签到按钮旁 | 显示连续签到统计 |
| 积分显示 | 数据统计区 | 当前积分余额 |
| 成就数量 | 数据统计区 | 已解锁成就数 |
| 称号展示 | 个人信息区 | 当前装备的称号 |
| 功能入口列表 | 页面主体 | 各功能入口 |

### 数据统计卡片

| 数据项 | 说明 |
|-------|------|
| 积分 | 当前可用积分 |
| 成就 | 已解锁成就数量 |
| 签到 | 连续签到天数 |
| 阅读 | 已阅读文章数 |

### 功能入口列表

| 入口 | 功能说明 |
|-----|---------|
| 签到日历 | 查看签到记录和日历 |
| 积分商城 | 兑换积分商品 |
| 我的成就 | 查看和管理成就 |
| 我的背包 | 查看背包物品 |
| 学习记录 | 查看阅读历史 |
| 站内消息 | 查看通知消息 |
| 诈骗反馈 | 提交诈骗举报 |
| 学生认证 | 绑定学生身份 |
| 设置 | 账户和系统设置 |

## 操作流程

### 修改头像

1. 点击当前头像
2. 选择新头像图片
3. 系统上传并更新
4. 显示"头像更新成功"

### 修改昵称

1. 点击昵称区域
2. 输入新昵称（2-20个字符）
3. 点击确认或回车
4. 昵称更新成功

### 每日签到

1. 点击"签到"按钮
2. 如果今日未签到：
   - 显示签到成功弹窗
   - 显示连续签到天数
   - 显示获得的积分
3. 如果今日已签到：
   - 显示"今天已签到"提示

### 查看签到日历

1. 点击"签到日历"入口
2. 进入签到日历页面
3. 查看当月签到情况
4. 已签到的日期打勾标记

### 查看消息通知

1. 点击"站内消息"入口
2. 或点击消息红点图标
3. 进入消息列表页

## 积分获取

在个人中心可以查看的积分来源：

| 来源 | 积分 | 说明 |
|-----|------|-----|
| 每日签到 | 5-50 | 连续签到奖励递增 |
| 阅读文章 | 2/篇 | 每日上限10篇 |
| AI对话 | 2/次 | 每日上限20次 |
| 答题正确 | 1/题 | 商城积分日上限30 |
| 成就解锁 | 各成就 | 额外积分奖励 |
| 学生认证 | 50 | 首次认证奖励 |

## 注意事项

- 首次进入会自动登录
- 用户信息同步到云端，换设备也可恢复
- 积分数据实时同步
- 未读消息会显示红点提示
