# 社区功能 - 用户手册

## 功能概述

社区是用户互动交流的平台，用户可以发布帖子、评论、关注其他用户，分享防骗经验和心得。

## 入口路径

- 点击底部导航栏"社区"图标

## 界面说明

### 社区主页元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 发帖按钮 | 右下角浮动 | 创建新帖子 |
| 帖子流 | 页面主体 | 展示帖子列表 |
| 加载更多 | 列表底部 | 上拉加载更多帖子 |

### 帖子卡片元素

| 元素 | 说明 |
|-----|------|
| 用户头像 | 点击可查看用户主页 |
| 用户昵称 | 发帖用户名称 |
| 用户称号 | 用户装备的称号徽章 |
| 发布时间 | 相对时间显示 |
| 帖子内容 | 帖子正文（截取预览） |
| 互动数据 | 点赞数、评论数 |

## 操作流程

### 浏览帖子

1. 进入社区页面，自动加载帖子列表
2. 向上滑动浏览更多帖子
3. 点击帖子卡片进入帖子详情

### 发布帖子

1. 点击右下角"+"发帖按钮
2. 进入发帖页面
3. 输入帖子内容（必填，最多500字）
4. 可选择添加图片（最多9张）
5. 点击"发布"按钮
6. 发布成功后自动返回社区列表

### 查看帖子详情

1. 点击帖子卡片进入详情页
2. 查看完整帖子内容
3. 可查看所有评论
4. 可进行点赞、评论操作

### 评论帖子

1. 在帖子详情页底部找到评论输入框
2. 输入评论内容
3. 点击发送按钮
4. 评论成功后显示在评论列表

### 关注用户

1. 点击帖子中的用户头像
2. 进入用户主页
3. 点击"关注"按钮
4. 按钮变为"已关注"状态

### 取消关注

1. 进入已关注用户的主页
2. 点击"已关注"按钮
3. 确认取消关注

## 注意事项

- 发布帖子需要登录
- 帖子内容需审核通过后才能公开显示
- 评论需遵守社区规范，禁止发布违规内容
- 帖子列表支持分页加载，每次加载10条
