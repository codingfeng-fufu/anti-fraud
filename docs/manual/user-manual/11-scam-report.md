# 诈骗反馈 - 用户手册

## 功能概述

诈骗反馈功能允许用户举报可疑的诈骗行为或信息，帮助其他用户提高警惕，同时协助反诈工作。

## 入口路径

- 个人中心 → 诈骗反馈

## 界面说明

### 表单元素

| 元素 | 类型 | 说明 |
|-----|------|-----|
| 诈骗类型 | 选择器 | 选择诈骗类型 |
| 诈骗渠道 | 选择器 | 选择诈骗发生渠道 |
| 诈骗账号 | 文本输入 | 填写可疑账号（可选） |
| 诈骗链接 | 文本输入 | 填写可疑链接（可选） |
| 详细描述 | 文本域 | 详细描述情况（必填） |
| 提交按钮 | 按钮 | 提交反馈 |

### 诈骗类型选项

- 电信诈骗
- 刷单返利
- 冒充客服退款
- 冒充公检法
- 校园贷/网贷
- 杀猪盘
- 其他

### 诈骗渠道选项

- 电话
- 短信
- 社交软件
- 网页/链接
- APP
- 线下
- 其他

## 操作流程

### 提交反馈

1. 进入诈骗反馈页面
2. 选择诈骗类型
3. 选择诈骗渠道
4. 填写诈骗账号（如有）
5. 填写诈骗链接（如有）
6. 填写详细描述（必填）
7. 点击"提交"按钮

### 提交成功后

1. 显示提交成功弹窗
2. 提示拨打96110反诈专线
3. 可选择：
   - 点击"拨打96110"立即拨打
   - 点击"稍后"关闭弹窗

## 注意事项

- 详细描述为必填项
- 提交后无法修改或撤回
- 如正在遭遇诈骗或已发生转账，**请立即拨打96110**
- 诈骗反馈仅用于信息收集，不代替报警

## 紧急情况处理

如遇以下情况，请立即拨打**96110**反诈专线：

- 正在遭受诈骗
- 已经完成转账
- 发现可疑人员接触
