# 设置与账户 - 用户手册

## 功能概述

设置页面提供账户管理、系统设置、关于信息等功能入口。

## 入口路径

- 个人中心 → 设置

## 界面说明

### 设置项目

| 项目 | 功能 |
|-----|------|
| 账户管理 | 管理账户信息和学生认证 |
| 清除缓存 | 清除小程序本地数据 |
| 意见反馈 | 提交使用反馈 |
| 关于我们 | 查看小程序信息 |
| 退出登录 | 退出当前账号 |

## 操作流程

### 账户管理

1. 点击"账户管理"
2. 进入账户管理页面
3. 可查看/修改：
   - 学生认证状态
   - 解除学生认证
   - 账户基本信息

### 清除缓存

1. 点击"清除缓存"
2. 确认清空操作
3. 清除本地存储的数据：
   - 阅读历史
   - 对话记录
   - 本地缓存
4. 显示"清除成功"提示

### 意见反馈

1. 点击"意见反馈"
2. 进入反馈页面
3. 输入反馈内容
4. 可添加截图
5. 提交反馈

### 关于我们

1. 点击"关于我们"
2. 查看小程序信息：
   - 版本号
   - 开发团队
   - 联系方式
   - 用户协议
   - 隐私政策

### 退出登录

1. 点击"退出登录"
2. 确认退出
3. 清除本地登录状态
4. 返回首页

## 注意事项

- 清除缓存会删除本地所有数据，谨慎操作
- 退出登录后云端数据仍保留
- 重新登录可恢复云端数据

## 隐私政策要点

- 用户数据加密存储
- 不向第三方泄露个人信息
- 用户可随时删除账户数据
- 详见《隐私政策》完整版
