# 成就系统 - 用户手册

## 功能概述

成就系统记录用户在反诈卫士中的各项成就，用户可以通过签到、学习、答题、对话等方式解锁成就，获得积分奖励和称号。

## 入口路径

- 个人中心 → 我的成就

## 界面说明

### 页面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 成就统计 | 页面顶部 | 已解锁/总成就数，完成进度 |
| 成就列表 | 页面主体 | 所有成就列表 |

### 成就卡片元素

| 元素 | 说明 |
|-----|------|
| 成就图标 | 成就徽章图标 |
| 成就名称 | 成就标题 |
| 成就描述 | 成就解锁条件 |
| 进度条 | 当前进度/目标值 |
| 积分奖励 | 解锁可获得的积分 |
| 称号奖励 | 部分成就解锁获得称号 |
| 展示按钮 | 已解锁成就可设为主页展示 |

## 操作流程

### 查看成就列表

1. 进入成就页面
2. 默认按解锁状态排序（已解锁在前）
3. 向下滑动查看所有成就

### 查看成就详情

1. 点击成就卡片
2. 查看成就解锁条件
3. 查看当前进度
4. 查看解锁奖励

### 设置主页展示

1. 找到已解锁的成就
2. 点击"展示"按钮
3. 该成就会在个人主页显示
4. 最多可设置6个成就展示

### 取消主页展示

1. 找到已设展示的成就
2. 再次点击"展示"按钮
3. 取消展示状态

## 注意事项

- 成就解锁后积分自动发放
- 部分成就解锁可获得称号，自动添加到称号列表
- 成就进度实时更新
- 连续签到中断后，签到天数重新计算
