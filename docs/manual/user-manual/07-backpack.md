# 背包系统 - 用户手册

## 功能概述

背包系统用于存放用户获得的所有虚拟物品，包括兑换的商品、道具卡、获得的奖励等。

## 入口路径

- 个人中心 → 我的背包
- 积分商城兑换成功后自动跳转

## 界面说明

### 页面元素

| 元素 | 位置 | 功能 |
|-----|------|-----|
| 分类标签 | 页面顶部 | 筛选物品类型 |
| 物品列表 | 页面主体 | 显示背包中的物品 |
| 统计信息 | 页面顶部 | 物品总数、未使用数 |

### 物品分类

| 分类 | 说明 |
|-----|------|
| 全部 | 显示所有物品 |
| 虚拟商品 | 话费、流量等虚拟奖励 |
| 道具卡 | 积分加倍卡等可使用道具 |
| 实体商品 | 已兑换的实物奖品 |

### 物品卡片元素

| 元素 | 说明 |
|-----|------|
| 物品图标 | 分类图标 |
| 物品名称 | 物品名称 |
| 物品状态 | 未使用/生效中/已使用/已过期/待发货 |
| 有效期 | 部分物品有有效期限制 |
| 使用按钮 | 部分物品可点击使用 |

## 物品状态说明

| 状态 | 说明 | 操作 |
|-----|------|-----|
| 未使用 | 物品可用 | 可使用 |
| 生效中 | 道具卡正在生效 | 查看详情 |
| 已使用 | 物品已消耗 | 无 |
| 已过期 | 超过有效期 | 无 |
| 待发货 | 实体商品等待发货 | 等待 |

## 操作流程

### 查看物品

1. 进入背包页面
2. 默认显示全部物品
3. 点击分类标签筛选

### 使用道具卡

1. 找到状态为"未使用"的道具卡
2. 点击"使用"按钮
3. 确认使用
4. 道具卡生效
5. 状态变为"生效中"

### 查看物品详情

1. 点击物品卡片
2. 弹出物品详情框
3. 查看物品说明和使用方法

## 物品类型详解

### 虚拟商品

- 话费充值：自动充值到绑定手机号
- 流量包：自动充值
- 会员权益：自动开通

### 道具卡

| 道具名称 | 效果 | 有效期 |
|---------|------|-------|
| 积分加倍卡 | 签到积分翻倍 | 24小时 |
| 经验加倍卡 | 阅读积分翻倍 | 24小时 |

### 实体商品

- 兑换后状态为"待发货"
- 需等待人工处理发货
- 发货后状态变为"已使用"

## 注意事项

- 部分物品有有效期，过期后将无法使用
- 道具卡使用后立即生效，请确认后使用
- 实体商品发货时间约3-7个工作日
- 如有问题请通过反馈渠道联系
