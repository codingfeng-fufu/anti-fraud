# 反诈卫士小程序项目根目录 (v3)

## 目录结构概览

- `app.js` - 小程序全局入口文件，负责云开发初始化、用户登录和全局数据管理
- `app.json` - 小程序全局配置文件，定义页面路径和窗口样式
- `app.wxss` - 小程序全局样式文件
- `sitemap.json` - 小程序页面索引配置
- `project.config.json` - 项目配置文件
- `pages/` - 页面文件目录
- `cloudfunctions/` - 云函数目录
- `utils/` - 工具函数目录
- `images/` - 图片资源目录
- `docs/` - 项目文档目录 (v1.1新增)

## 重要说明

**每当所属的代码发生变化时，必须对相应的文档进行更新操作！**

此项目为反诈骗教育类微信小程序。v3版本在原有“资讯/AI助手/积分/成就/称号”基础上新增趣味答题（题库/判分/排行榜/每日奖励）、社区、评论与回复、个人主页、关注与站内消息、搜索、诈骗反馈，以及积分商城兑换逻辑修复与话费/流量类商品（人工发放记录）。
