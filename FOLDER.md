# 反诈卫士小程序项目根目录 (v1.1)

## 目录结构概览

- `app.js` - 小程序全局入口文件，负责云开发初始化、用户登录和全局数据管理
- `app.json` - 小程序全局配置文件，定义页面路径和窗口样式
- `app.wxss` - 小程序全局样式文件
- `sitemap.json` - 小程序页面索引配置
- `project.config.json` - 项目配置文件
- `pages/` - 页面文件目录
- `cloudfunctions/` - 云函数目录
- `utils/` - 工具函数目录
- `images/` - 图片资源目录
- `docs/` - 项目文档目录 (v1.1新增)

## 重要说明

**每当所属的代码发生变化时，必须对相应的文档进行更新操作！**

此项目为反诈骗教育类微信小程序，v1.1版本新增了称号系统、成就系统升级、多称号佩戴、用户管理优化等功能，包含用户管理、签到积分、成就系统、积分商城、文章阅读、AI对话等功能。
