# 页面目录 (v3)

## 目录结构概览

- `index/` - 首页页面，展示反诈资讯和功能入口
- `community/` - 社区页面（v3新增），帖子流与社区入口（Tab页）
- `community-post-create/` - 发帖页面（v3新增）
- `community-post-detail/` - 帖子详情页面（v3新增），含评论区
- `chat/` - AI助手对话页面，提供反诈咨询服务
- `user/` - 个人中心页面，用户信息管理和功能入口，v1.1新增称号展示
- `messages/` - 站内消息页面（v3新增），关注用户发帖/评论回复提醒
- `sign-in/` - 每日签到页面，展示签到日历与自动签到功能
- `my/` - 我的页面，通过昵称入口展示用户基本信息、成就与称号
- `profile/` - 个人主页页面（v3新增），用于查看他人资料（称号/成就展示位/答题积分/关注）
- `settings/` - 设置页面
- `points/` - 积分商城页面，积分兑换功能，v1.1新增称号兑换专区
- `redeem-info/` - 兑换信息填写页面（v3新增），用于话费/流量等人工发放商品填写手机号/运营商
- `logs/` - 日志页面
- `learning/` - 学习记录页面
- `feedback/` - 帮助与反馈页面
- `scam-report/` - 诈骗反馈页面（v3新增），提交诈骗行为并提示拨打96110
- `bind-student/` - 学生信息绑定页面
- `article-detail/` - 文章详情页面
- `achievements/` - 成就页面，展示用户获得的成就，v1.1新增称号奖励展示
- `quiz/` - 趣味答题页面（v3新增），随机抽题、答题判分与积分奖励
- `quiz-leaderboard/` - 答题排行榜页面（v3新增）
- `search/` - 搜索页面（v3新增），搜索资讯与用户
- `about/` - 关于我们页面
- `title-management/` - 称号管理页面 (v1.1新增)，用于管理用户称号佩戴

## 重要说明

**每当所属的代码发生变化时，必须对相应的文档进行更新操作！**

此目录包含小程序的所有页面文件，每个页面通常包含 `.js`、`.wxml`、`.wxss` 和 `.json` 四个文件，分别对应逻辑、结构、样式和配置。v3版本新增了趣味答题、社区、评论、个人主页、关注与站内消息、搜索、诈骗反馈等页面。
