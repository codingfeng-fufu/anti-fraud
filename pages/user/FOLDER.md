# 用户管理页面目录 (v3)

## 文件概览

- `user.js` - 页面逻辑文件，处理用户信息、签到、积分、称号等功能
- `user.wxml` - 页面结构文件，定义用户界面布局，v1.1新增称号展示区域，v3新增站内通知入口
- `user.wxss` - 页面样式文件，定义用户界面外观，v1.1新增称号样式
- `user.json` - 页面配置文件，配置页面窗口样式等

## 功能说明

用户管理页面是小程序的个人中心模块，提供用户信息展示与编辑、签到功能、积分和成就统计、称号展示与管理入口等功能。v3新增站内通知入口（右上角铃铛），用于进入消息页面并展示未读数。
