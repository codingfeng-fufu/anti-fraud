# 用户管理页面目录 (v1.1)

## 文件概览

- `user.js` - 页面逻辑文件，处理用户信息、签到、积分、称号等功能
- `user.wxml` - 页面结构文件，定义用户界面布局，v1.1新增称号展示区域
- `user.wxss` - 页面样式文件，定义用户界面外观，v1.1新增称号样式
- `user.json` - 页面配置文件，配置页面窗口样式等

## 功能说明

用户管理页面是小程序的个人中心模块，v1.1版本新增称号系统，提供用户信息展示与编辑、签到功能、积分和成就统计、称号展示与管理入口等功能。用户昵称可跳转至“我的”页面查看详细资料，签到入口进入签到日历页面。页面采用现代化设计风格，使用渐变色背景和卡片式布局。
