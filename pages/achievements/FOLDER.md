# 成就页面目录 (v3)

## 文件概览

- `achievements.js` - 页面逻辑文件，处理成就数据、解锁状态检查、称号奖励展示等功能，v3新增成就展示位设置
- `achievements.wxml` - 页面结构文件，定义成就界面布局，v1.1新增称号奖励展示
- `achievements.wxss` - 页面样式文件，定义成就界面外观，v1.1新增称号奖励样式
- `achievements.json` - 页面配置文件，配置页面窗口样式等

## 功能说明

成就页面展示用户的各项成就，包括签到类、学习类、对话类、积分类以及v3新增的答题类成就。用户可在成就页面将已解锁成就设置为“主页展示位”（最多6个），用于个人主页展示。
