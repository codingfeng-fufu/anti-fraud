# TabBar 图标说明

## 需要的图标文件

在这个文件夹中需要放置以下 6 个图标文件：

### 图标列表

1. **home.png** - 首页图标（未选中）
   - 建议尺寸：81px × 81px
   - 颜色：灰色 (#94a3b8)

2. **home-active.png** - 首页图标（选中）
   - 建议尺寸：81px × 81px
   - 颜色：蓝色 (#0ea5e9)

3. **chat.png** - AI助手图标（未选中）
   - 建议尺寸：81px × 81px
   - 颜色：灰色 (#94a3b8)

4. **chat-active.png** - AI助手图标（选中）
   - 建议尺寸：81px × 81px
   - 颜色：蓝色 (#0ea5e9)

5. **user.png** - 我的图标（未选中）
   - 建议尺寸：81px × 81px
   - 颜色：灰色 (#94a3b8)

6. **user-active.png** - 我的图标（选中）
   - 建议尺寸：81px × 81px
   - 颜色：蓝色 (#0ea5e9)

## 临时解决方案

如果暂时没有图标，可以：

1. 从 [iconfont.cn](https://www.iconfont.cn/) 下载免费图标
2. 使用微信开发者工具的默认图标
3. 使用 AI 工具（如 Midjourney）生成简单图标

## 图标设计建议

- 使用简洁的线条风格
- 保持图标一致性
- 确保在小尺寸下清晰可辨
- 未选中状态使用灰色
- 选中状态使用品牌蓝色
